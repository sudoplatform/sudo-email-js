ListEmailAddressesResult

ListEmailAddressesResult: ListOperationResult<EmailAddress, SealedEmailAddressProps>
  • Defined in src/public/typings/listOperationResult.ts:19

The result object for list email addresses APIs.

  • On success, contains the list of requested email addresses.
  • On partial success, contains the list of email addresses that were fetched and unsealed successfully as well as the list of email addresses that could not be unsealed successfully and the error indicating why unsealing failed. An email address may fail to unseal if the client version is not up to date or the required cryptogrpahic key is missing from the client device.
  • On failure, contains an error object indicating why the list operation failed.
